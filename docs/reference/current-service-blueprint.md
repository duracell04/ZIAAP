## Current Contract Alignment projection

Pilot 01 uses only the current I0 wedge: immutable contract version, selected
clauses, approved scenarios, sealed independent responses, party confirmation,
symmetric reveal, resolution or conscious openness, legal-status selection and
version lock. Every meaningful transition requires an attributable human act.

Original exploratory text and AI-normalised text remain separate. The model
cannot reveal responses, change authoritative state, assign legal status or
lock a version. Results default to non-binding unless authorised parties later
complete and evidence a separate external adoption act.

## Public projection

Align maps to Gate 1 and Test scenarios maps to Gate 2. Both represent the
current wedge. Structure dispute maps to Gate 4, Human review maps to Gate 5
and Procedural record maps to Gate 6; all three are visibly future extensions.
Gate 3 is not implied to be part of the current public product.

The public path excludes appointment interaction and all live execution.

## Failure and recovery

- Incomplete alignment blocks guided progression.
- Stale confirmation, authority failure, withdrawal, incident or contract
  version change blocks or invalidates the affected transition.
- A failed normalisation returns to party correction; it never becomes a
  confirmed position automatically.
- Comparison cannot be revealed until both parties confirm or correct their
  representations.
- Constitution changes invalidate scenario and downstream artifacts.
- Configuration-hash mismatch blocks the fictional appointment transition.
- Live execution failure remains visible and never becomes illustrative
  success.
- Missing evidence blocks deterministic conclusions or escalates to human
  review.
- Future mediation content stays isolated from merits production unless an
  object is explicitly authorised for export.
- A reasoning memorandum cannot appear before the fictional preliminary
  assessment.
- A simulated human record requires disposition, rationale and checklist.

## Current boundaries

Every action is synthetic, simulation-only and without legal effect. The C0
service does not verify identity, accept a confidential matter, execute Pilot
01, recruit participants, run a pricing experiment, appoint an arbitrator,
mediate, issue an award, prove enforceability or implement partner ownership.
