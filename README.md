# ZIAAP

Zero-Instance Algorithmic Arbitration Protocol.

This repository currently contains only the lean scaffold for a hackathon
prototype. Application behavior, documentation content, framework setup, AI,
persistence, authentication, and deployment are not implemented yet.

## Repository map

- `docs/` — seven product, governance, experience, technical, and brand documents
- `app/` — Next.js pages and API route placeholders
- `components/` — reusable interface component placeholders
- `lib/` — application and domain-logic placeholders
- `data/` — the single demo-case placeholder
- `public/` — public brand assets and demo documents
